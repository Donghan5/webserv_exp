About Our Project

This is a custom HTTP server implementation designed to provide advanced web hosting capabilities while maintaining high performance and security standards.

Our Webserv project supports:

  • Multiple CGI handling for Python, PHP, Perl, and Bash scripts
  • Secure file uploads with comprehensive validation
  • Session management for stateful applications
  • Custom error pages for improved user experience
  • Directory listing with configurable options
